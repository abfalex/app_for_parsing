# Парсинг матчей

Этот проект позволяет собрать информацию о футбольных матчей с сайта [flashscore](https://www.flashscorekz.com/), структурируя всё в таблицу Excel.

### Установка
Перед установкой убедитесь, что у вас установлен [Python 3.11](https://www.python.org/)

Для работы с проектом его необходимо установить или клонировать:

```bash
git clone https://github.com/abfalex/app_for_parsing.git
```

Далее вам необходимо создать виртуальную среду для удобного использования (рекомендуется):

   ```bash
   python -m venv <имя_венва>
   ```

После установки виртуальной среды ее необходимо активировать:

  - В Windows:

     ```bash
     <venv_name>\Scripts\activate
     ```

- В macOS и Linux:

     ```bash
     source <venv_name>/bin/activate
     ```

После вам необходимо установить необходимые библиотеки. Введите эту команду в терминал:

```bash
pip install -r requirements.txt
```

### Запуск
Укажите желаемые параметры и вид спорта:

```python
STAT_PARAMS = ["Выиграно на подаче", "Выиграно очков у сетки"] # 19 строчка кода
sport = "ТЕННИС" # 112 строчка кода
```

Запустить парсинг:

```bash
python main.py
```

Результатом парсинга окажется Excel-файл, сохранившийся в корневой папке проекта.
